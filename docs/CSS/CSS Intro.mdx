# CSS : Cascading Style Sheet

## CSS Selector

\*\* In CSS selector is used to apply specific style to an element, selector is a pattern or a name in which we define
some styles, and this selector is bound with element to apply that styles to an element. Like class selector, universal selector.

- We will learn more about selector in CSS : [Link to CSS]

```
// Universal selector
* {
    color:orange
}
//

// Pseudo selector
a:hover {
color:blue
}

```
